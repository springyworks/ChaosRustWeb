# ChaosRustWeb - 4D Wave Ripple Tank Simulation

A real-time 4D wave equation simulation running in the browser via WebAssembly, powered by Bevy 0.15 and Burn 0.20.

## Live Demo

[View Live Demo on GitHub Pages](https://yourusername.github.io/ChaosRustWeb/)

## What This Demonstrates

- **5,184 Interactive Particles**: Three coupled 12x12x12 tensor grids (1,728 particles each)
- **Wave Physics**: FDTD method for wave equation with Laplacian operator
- **Inter-Tensor Coupling**: Ring topology (Tensor 0 -> 1 -> 2 -> 0) with 150.0 coupling strength
- **Real-Time Graphics**: Bevy 0.15 rendering in browser via WebGL2
- **Burn Integration**: NdArray backend for synchronous tensor operations in WASM
- **Interactive Controls**: Mouse-based camera manipulation and keyboard shortcuts

## Features

### Physics Simulation
- Wave speed: 17.0 units
- Damping: 0.95 (energy dissipation)
- Pinger system: Creates wave impulses every 2 seconds, rotating through tensors
- Energy tracking: Real-time kinetic and potential energy display

### Interactive Controls
- **Left Mouse Drag**: Orbit camera around scene
- **Right Mouse Drag**: Pan camera laterally
- **Mouse Wheel**: Zoom in/out
- **Space Bar**: Toggle auto-rotate on/off

### Visualization
- Color-coded particle grids (Red, Green, Blue)
- Velocity-scaled particle rendering with emissive glow
- On-screen stats: Kinetic energy, potential energy, pinger status

## Running Locally

### Prerequisites
- [Rust](https://rustup.rs/) (latest stable)
- [Trunk](https://trunkrs.dev/) (`cargo install trunk`)
- `wasm32-unknown-unknown` target (`rustup target add wasm32-unknown-unknown`)

### Build and Run

```bash
# Development server with hot reload
trunk serve --open

# Production build
trunk build --release
```

Open http://127.0.0.1:8080 in your browser.

## Technical Architecture

### Technology Stack
- **Bevy 0.15**: Game engine and ECS framework
- **Burn 0.20**: Tensor library with NdArray backend for WASM compatibility
- **WebGL2**: Graphics API for cross-platform browser rendering
- **Trunk**: WASM build tool and development server

### Project Structure
- `src/main.rs`: Complete simulation with comprehensive Rust documentation
- `Cargo.toml`: Dependency configuration for WASM target
- `index.html`: Web application entry point
- `dist/`: Build output (generated by Trunk)

### Physics Implementation
- 3D Laplacian stencil operator for wave propagation
- Verlet integration for stable particle dynamics
- Ring-coupled tensor networks for complex wave patterns
- Position-based damping and inter-tensor force exchange

## Performance
- 5,184 particles updated at 60 FPS
- Synchronous tensor operations (no GPU async overhead)
- Optimized for browser rendering with WebGL2

## Development Notes
- Uses local Bevy build with webgl2 features
- NdArray backend chosen over WGPU for WASM (avoids async GPU readback)
- MessageReader API for Bevy 0.15 input handling
- Comprehensive inline documentation for all major components
