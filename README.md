# ChaosRustWeb

A Bevy + Burn integration demo running in the browser via WebAssembly + WGPU/WebGL2.

## What This Demonstrates

- **Bevy 0.15** 3D rendering in the browser
- **Burn 0.20** Tensor operations running on the backend (WGPU/CubeCL)
- Interactive 3D scene with rotating objects
- Real-time color updates driven by Burn tensor operations

## Running Locally

Prerequisites:
- [Rust](https://rustup.rs/)
- [Trunk](https://trunkrs.dev/) (`cargo install trunk`)
- `wasm32-unknown-unknown` target (`rustup target add wasm32-unknown-unknown`)

Run the development server:

```bash
trunk serve --open
```

Open http://127.0.0.1:8080 in your browser.

## Features

- **3D Scene**: Red Cube, Green Sphere, Blue Cylinder, Yellow Torus.
- **Animation**: Objects rotate, camera orbits the scene.
- **AI/Tensor**: The "Burn Target" (Green Sphere) changes color based on random tensors generated by Burn on the GPU.

## Project Structure

- `src/main.rs`: Main application entry point, scene setup, and Burn integration logic.
- `Cargo.toml`: Dependencies (Bevy, Burn, Wasm-bindgen).
- `assets/`: Static assets (if any).
- `index.html`: Web entry point.

## Notes

- Uses `burn` with `wgpu` backend (via `burn-wgpu` / `CubeCL`).
- Configured for WebGL2/WebGPU compatibility.
