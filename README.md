# Bevy Web Test

Simple Bevy example running in the browser via WebAssembly + WebGL2.

## What This Demonstrates

- 3D rendering with Bevy 0.14 in the browser
- Rotating objects (cubes, sphere, ground plane)
- Camera orbit animation
- Emissive materials for glow effect
- WebGL2 backend

## Running Locally

```bash
trunk serve
```

Open http://127.0.0.1:8080
